# AI Resume Analyzer
The AI Resume Analyzer is a web-based application developed using Python and Flask. This system allows users to upload their resume in PDF format and automatically analyzes the resume to detect technical skills. It calculates a resume score, displays the skill match percentage, and suggests additional skills that can improve the resume.
This project demonstrates how Artificial Intelligence concepts can be used to evaluate resumes and help candidates improve their profiles.

# Features:
Upload resume in PDF format
Automatic text extraction from resume
Skill detection from resume content
Resume score calculation
Skill match percentage
Suggested skills for improvement
Simple and attractive user interface

# Technologies Used:
Python
Flask
HTML
CSS
PyPDF2
